My Projects


-User Details-

Valet Manager is a website built for keeping track of data on the ongoings of a valet operation. Users can save data on their valet employees and locations where they operate. After registering an account, users can either Add Location or Add Valet. Completing these forms will populate the Locations and Valets pages with the updated data.

When there is both a Location and a Valet saved, the user can assign the two to each other by clicking Details on either a Location or a Valet. Scrolling down on this page will reveal a dropdown menu populated with options. Selecting one of these will populate the area below the Details page with the relative data.

Valet Manager

A website that keeps track of your valet business

View Project Details

-Tech Details-


Valet Manager has login-logout functionality, complete with JWT and cookie. Registered user passwords are hashed with Bcrypt and stored on MongoDB. Express is used to route files together in the back end. Mongoose _id's used to associate Locations and Valets with each other.

Valet Manager was developed by Ryan Kenyon as part of Kingsland University's Full Stack Developer program.

ForumAnon is a simple website for sharing thoughts and ideas. After registering, users can either create new threads or add comments to existing ones. Users are able to edit their own threads/comments after posting, but unable to edit threads/comments of others.

ForumAnon

A simple forum for hosting and sharing ideas

View Project Details

-Tech Details-

ForumAnon was built with a Handelbars and uses a MongoDB hosted server to host data. Login-logout functionality with JWT and cookie. MongoDB _id associated with posts/comments to prevent unauthorized editing or deletion.

ForumAnon was built by Ryan Kenyon and Binh Tran as part of Kingsland University's Full-Stack Developer program.

WeatherApp is a simple website that searches for current and forecasted weather from any zip code. Data includes highs and lows, humidity, and weather messages. Forecasted weather is displayed for the next seven days below the current weather.

Weather App

A simple app demonstrating API fetch requests.

View Project Details

-Tech Details-

WeatherApp is a simple HTML page demonstrating a successful fetch request to a weather website's API. After validating the zip code, the app will make a fetch request to openweathermap.org and display the returned data through DOM manipulation. If zip code is invalid, error page will be displayed.

WeatherApp was developed by Ryan Kenyon and Binh Tran as part of Kingsland University's Full Stack Developer program.
