About Valet Manager

User Details

Valet Manager is a website built for keeping track of data on the ongoings of a valet operation. Users can enter information on valet employees and they locations they operate at.

Tech Details

Valet Manager was built using ReactJS and Mongo-DB. Registering an account will hash password with Bcrypt, and logging in will create a JWT and cookie respectively.
